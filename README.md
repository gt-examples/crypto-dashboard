# Crypto Dashboard

A cryptocurrency portfolio and market dashboard with full internationalization support.

**[Live Demo](https://crypto-dashboard.generaltranslation.dev)** | **[General Translation Docs](https://generaltranslation.com/docs)**

## About

A realistic crypto dashboard demonstrating GT's translation capabilities including formatted currencies, numbers, dates, and full UI translation across English, Spanish, and Japanese.

## GT Features Used

- `<T>` — JSX translation
- `<Currency>` — Locale-aware currency formatting
- `<Num>` — Number formatting
- `<DateTime>` — Date/time formatting
- `<LocaleSelector>` — Language picker
- `useGT` — String translations
- `loadTranslations` — Local translation storage

## Getting Started

```bash
git clone https://github.com/gt-examples/crypto-dashboard.git
cd crypto-dashboard
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
